更改hexo博客主题

更换主题

1.首先在浏览器https://hexo.io/themes/进入github官方主题界面，如图：

i

2.里面有非常多的主题，选择一个自己喜欢的，点进去 ，就比如说我选了这个：

i

3.记住这个主题的名字 叫cyanstyle ,点击去之后就会进入这个主题拥有者的博客

4.然后找到博主的仓库，进入仓库；在搜索栏中搜索刚才那个主题的名字 咱这里是cyanstyle:

i

5.点进去这篇博客，将它的下载地址复制下来：

i

之后

1.先复制仓库下载主题 （在git中）

$ hexo clone https://github.com/wizardforcel/hexo-theme-cyanstyle.git themes/cyanstyle

2.下载完后会在themes目录下生成一个名为cyanstyle文件

3.更改一下hexo项目目录下的配置文件_config.yml，主题名修改一下：

theme： cyanstyle

4.然后清空一下public缓存：

$ hexo clean

5.然后重新生成datebase 静态文件：

$ hexo g

6.之后打开本地仓库服务，观察主题是否有变化 localhost:4000

7.在发现主题更改之后，上传到远程仓库就行了：

$ hexo d
